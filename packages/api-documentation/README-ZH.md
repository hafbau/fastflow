# Fastflow API 文档

[English](./README.md) | 中文

这是 Fastflow 公共 API 的列表，允许用户以编程方式执行与 GUI 中相同的许多任务。

独立运行：

1. 启动 Fastflow 服务器：
    ```sh
    cd Fastflow
    pnpm start
    ```
2. 启动 API 文档服务器：
    ```sh
    cd packages/api-documentation
    pnpm start
    ```

开发模式下运行：

```sh
cd Fastflow
pnpm dev
```

文档将在此地址提供：http://localhost:6655/api-docs
